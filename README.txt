This is Full Duplex based Real Time Chat Program. Here Port Number is given as an argument, So don't forget to pass the port number while running the code.
 
   Follow these steps to run the code:
   1. run the server code first: gcc server.c -o server
   2.  ./server port_number
   3. run the client code:  gcc client.c -o client
   4.  ./client IpAddress port_number
